what are the ways of getting code from github?

We have 4 ways required for getting code from github.
They are:
1. Download as ZIP: Go to repo → Code → Download ZIP → Extract.
2.Clone using Git (Command Line):Install Git → Go to repo → Code → Copy URL → Terminal(git clone url)
3. Fork and Clone:Log in → Go to repo → Fork → Clone your fork (git clone )
4.GitHub Desktop:Install GitHub Desktop → Go to repo → Code → Open with GitHub Desktop → Clone.

know the d/f b/w the all type of getting code?

Download as ZIP: Get the code quickly without using Git or tracking changes.Clone with Git: Download the code ,keep it synced with updates using Git commands. Fork and Clone: Create your own copy of the repo to modify and contribute back.GitHub Desktop: Use a graphical app to manage the code and Git without typing commands.
